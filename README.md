# HR Engagement Analytics Dashboard

## Project Overview
A comprehensive HR analytics dashboard built with 
Microsoft Power BI analyzing 15,000 employee records 
across 6 departments and multiple geographic regions.

## Dashboard Pages
- Executive Overview — Workforce KPIs & attrition
- Talent & Retention — Risk profiling & star employees
- Employee Wellbeing — Sensor data & health metrics
- Remote vs On-Premise — Work arrangement analysis

## Key Findings
- 24% attrition rate (above industry benchmark)
- 4,742 high-risk employees (31.6% of workforce)
- Remote employees show higher satisfaction & performance
- 3,998 star employees identified for retention programs

## Tools & Skills
- Microsoft Power BI
- Power Query (data cleaning)
- DAX (10 calculated columns, 8 measures)
- Data Storytelling & Report Writing

## Dataset
HR Engagement Sat Sales — 15,000 rows, 59 original columns
